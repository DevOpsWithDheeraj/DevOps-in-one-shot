# ğŸ§± **4. Install Kubernetes using KIND (Kubernetes in Docker)**

### ğŸ“Œ **4.1 Requirements**

### ğŸ“¥ **4.2 Install KIND**

```

### ğŸ—ï¸ **4.3 Create a Cluster**

```

### ğŸ” **4.4 Verify Nodes**

```bash

```

### ğŸ“Œ **4.5 Multi-node Cluster (Example)**

`kind-config.yaml`:

```yaml

apiVersion: kind.x-k8s.io/v1alpha4
nodes:
  - role: control-plane
  - role: worker
  - role: worker
```

Create cluster:

```bash
kind create cluster --config kind-config.yaml
```
